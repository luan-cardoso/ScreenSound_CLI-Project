# ScreenSound

Aplicação de linha de comando (CLI) desenvolvida em **C# / .NET** para gerenciamento de um catálogo de músicas e artistas. Projeto desenvolvido como estudo introdutório da linguagem C#.


## 📋 Sobre o projeto

O ScreenSound permite cadastrar, listar e gerenciar artistas e suas músicas diretamente pelo terminal. A aplicação é organizada com menus interativos e segue uma estrutura orientada a objetos com modelos bem definidos.


## 🗂️ Estrutura do projeto

```
ScreenSound/
├── Menus/          # Classes responsáveis pelos menus interativos do CLI
├── Modelos/        # Modelos de domínio (ex: Artista, Música)
├── Program.cs      # Ponto de entrada da aplicação
└── ScreenSound.csproj
ScreenSound.sln     # Arquivo de solução do Visual Studio
```

---

## 🚀 Como executar

### Pré-requisitos

- [.NET SDK](https://dotnet.microsoft.com/download) 6.0 ou superior

### Passos

```bash
# Clone o repositório
git clone https://github.com/luan-cardoso/ScreenSound_CLI-Project.git

# Acesse a pasta do projeto
cd ScreenSound_CLI-Project/ScreenSound

# Execute a aplicação
dotnet run
```
